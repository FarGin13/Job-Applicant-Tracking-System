# Job Applicant Tracking System

A simple **Job Applicant Tracking System** built with **PHP (backend)**, **MySQL (database)**, and **Bootstrap (UI)**.  
It supports user authentication for two roles (Job Advertiser / Candidate), job posting by advertisers, and a jobs listing page.

## Features
- Register as **Job Advertiser** or **Candidate**
-  Sign in / Sign out (PHP sessions)
-  Job Advertiser can **Post a Job**
-  View all jobs in a table (Jobs page)

## Tech Stack
- PHP (server-side)
- MySQL (database)
- Bootstrap 5 (frontend UI)
